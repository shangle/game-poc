Welcome to the official **Retro Engine Studio** documentation. This suite is a modern, browser-native reimagining of classic 2.5D game engines.

## What is Retro Engine Studio?

It is a zero-backend, high-performance game creation toolkit. By leveraging **Three.js** for rendering and **Web Components** for UI, it provides a seamless workflow for designing, testing, and sharing retro-style FPS experiences.

### Key Philosophy
1.  **AI-Native:** Designed for Large Language Models to generate level data effortlessly.
2.  **No-Code Sharing:** Entire levels are stored in the URL hash via Base64.
3.  **Modular:** Native Web Components ensure the editor can be embedded anywhere.
